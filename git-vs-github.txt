# O que é o Git? :computer:

- Git é um sistema de controle de versão, usado para controlar as alterações dos arquivos, criação de novas pastas e para transitarmos por entre as pastas criadas.

# Principais comandos do Git: :computer_mouse:

- **MKDIR -** Cria Diretório (pasta)

- **CD -** Abre Diretório

- **CD . . -** Volta para o diretório anterior

- **GIT INIT -** Cria repositório local (no pc)

- **GIT ADD -** Adiciona alterações no repositório

- **GIT COMMIT -** Confirma e salva as alterações

- **GIT PUSH -** Sobe alterações locais para o repositório remoto

- **GIT STATUS -** Confere se existe alterações na branch

  


# O que é o GitHub? :cloud:

- o GitHub é basicamente uma nuvem que hospeda um sistema de controle de versão Git. Ele permite que os desenvolvedores colaborem e façam mudanças em projetos compartilhados. Usado também como portfólio
